# Visão Geral dos Comandos do Metasploit Framework

## Comandos Principais

- **?**: Menu de ajuda
- **background** / **bg**: Coloca a sessão atual em segundo plano
- **bgkill**: Encerra um script de meterpreter em segundo plano
- **bglist**: Lista os scripts de meterpreter em execução em segundo plano
- **bgrun**: Executa um script de meterpreter como uma thread em segundo plano
- **channel**: Exibe informações ou controla canais ativos
- **close**: Fecha um canal
- **detach**: Desconecta a sessão de meterpreter (para http/https)
- **disable_unicode_encoding**: Desativa a codificação de strings unicode
- **enable_unicode_encoding**: Ativa a codificação de strings unicode
- **exit** / **quit**: Termina a sessão de meterpreter
- **get_timeouts**: Obtém os valores atuais de tempo limite da sessão
- **guid**: Obtém o GUID da sessão
- **help**: Menu de ajuda
- **info**: Exibe informações sobre um módulo Post
- **irb**: Abre um shell interativo Ruby na sessão atual
- **load** / **use**: Carrega uma ou mais extensões de meterpreter
- **machine_id**: Obtém o ID do MSF da máquina conectada à sessão
- **pry**: Abre o debugger Pry na sessão atual
- **quit**: Termina a sessão de meterpreter
- **read**: Lê dados de um canal
- **resource**: Executa comandos armazenados em um arquivo
- **run**: Executa um script de meterpreter ou módulo Post
- **secure**: (Re)Negocia a criptografia de pacotes TLV na sessão
- **sessions**: Alterna rapidamente para outra sessão
- **set_timeouts**: Define os valores atuais de tempo limite da sessão
- **sleep**: Força o Meterpreter a ficar em silêncio e depois restaura a sessão
- **transport**: Gerencia os mecanismos de transporte
- **uuid**: Obtém o UUID da sessão atual
- **write**: Escreve dados em um canal

## Stdapi: Comandos do Sistema de Arquivos

- **cat**: Exibe o conteúdo de um arquivo na tela
- **cd**: Altera o diretório
- **checksum**: Recupera o checksum de um arquivo
- **chmod**: Altera as permissões de um arquivo
- **cp**: Copia origem para destino
- **del**: Exclui o arquivo especificado
- **dir** / **ls**: Lista arquivos
- **download**: Baixa um arquivo ou diretório
- **edit**: Edita um arquivo
- **getlwd**: Imprime o diretório de trabalho local (alias para lpwd)
- **getwd**: Imprime o diretório de trabalho
- **lcat**: Exibe o conteúdo de um arquivo local na tela
- **lcd**: Altera o diretório de trabalho local
- **ldir** / **lls**: Lista arquivos locais
- **lmkdir**: Cria um novo diretório na máquina local
- **lpwd**: Imprime o diretório de trabalho local
- **mkdir**: Cria diretório
- **mv**: Move origem para destino
- **pwd**: Imprime o diretório de trabalho
- **rm**: Exclui o arquivo especificado
- **rmdir**: Remove diretório
- **search**: Procura por arquivos
- **upload**: Carrega um arquivo ou diretório

## Stdapi: Comandos de Rede

- **arp**: Exibe o cache ARP do host
- **ifconfig** / **ipconfig**: Exibe interfaces de rede
- **portfwd**: Encaminha uma porta local para um serviço remoto
- **resolve**: Resolve um conjunto de nomes de host no alvo
- **route**: Visualiza e modifica a tabela de roteamento

## Stdapi: Comandos do Sistema

- **execute**: Executa um comando
- **getenv**: Obtém um ou mais valores de variáveis de ambiente
- **getpid**: Obtém o identificador de processo atual
- **getuid**: Obtém o usuário sob o qual o servidor está sendo executado
- **kill**: Encerra um processo
- **localtime**: Exibe a data e hora local do sistema de destino
- **pgrep** / **pkill**: Filtra e encerra processos pelo nome
- **ps**: Lista processos em execução
- **shell**: Acessa um shell de comando do sistema
- **sysinfo**: Obtém informações sobre o sistema remoto, como SO

## Stdapi: Comandos de Saída de Áudio

- **play**: Reproduz um arquivo de áudio de forma de onda (.wav) no sistema de destino
